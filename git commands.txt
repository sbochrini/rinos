Με τα παραπάνω θα έχεις στο pc ακριβώς ότι στο github repository. Αν κάνεις αλλαγές στο pc, αφού κάνεις τις αλλαγές, οι βασικές εντολές είναι:

List the files you've changed and those you still need to add or commit: git status

Add one files to staging (index): git add <filename>
or more files to staging (index): git add *	

Commit any files you've added with git add, and also commit any files you've changed since then: git push origin master
Εδώ σε αναγκάζει να γράψεις και ένα μήνυμα για το commit, ώστε να ξέρεις γιατί κάνει commit. Στο παράθυρο που θα ανοίξει, πατάς i για να ξεκινήσει να γράφει, και όταν γράψεις το μήνυμα πατάς το ESC και μετά με πατητό το shift πατάς ZZ	git commit -a
Send changes to the master branch of your remote repository:
